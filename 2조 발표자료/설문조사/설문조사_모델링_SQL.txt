설문조사_모델링_SQL
[ 요구 분석서 ]
설문조사를 실시하는 설문 사이트.

1. 설문 목록
  1) 로그인 상태아니어도 보기 가능
  2) 목록번호 -> 설문지 등록 순서대로 번호를 자동으로 출력
  3) 질문, 작성자, 시작일/종료일, 항목수, 참여자수, 상태(진행중/완료) 표시

2. 설문지 작성
  1) 관리자만 설문지 작성 가능하도록 설정
  2) 일반 회원일 경우 접근 불가하도록 설정
  2) 설문 질문 작성,
	시작/종료일 설정 ( 연도/월/일 ),
	항목수(=선택지 개수) 설정,
	항목 작성				=> insert 문 사용해서 기입
  3) 설문 질문은 의미가 명확해야함.
    항목수를 한정적이므로 객관적으로 선택할 수 있도록 폐쇄형 질문으로 설정
  3) 확인 -> 설문지 작성 완료시 설문 목록에 추가 후 설문 목록 보여줌.
      취소 -> 설문 목록으로 돌아가도록 함

2-1. 설문지 수정
  1) 현재 날짜 기준으로 설문완료시기 이전일 경우
     시작/종료일 설정을 수정 가능하도록 설정.
  2) 질문/항목수/항목 변경

3. 투표하기
  1) 로그인 상태인지 확인
  2) 현재 날짜 기준으로 설문이 가능한 기간인지 확인
  3) 설문 상태가 진행중(기간내)일 경우 투표하도록 함
  4) 설문에 따른 항목을 선택함 -> 선택한 항목을 insert문 사용해서 내용 기록
  5) 확인 선택 -> 결과를 표시 / 취소 선택 -> 다시 목록으로 돌아가도록

3-1. 투표결과 확인
  1) 총 참여자 수, 질문, 항목, 선택비율(%)로 구성
  2) 투표에 따른 결과를 막대그래프로 표시함

4. 설문 보기
  1) 로그인 상태 아니어도 보기 가능
  2) 설문목록 중 선택해서 보기 -> where문 사용해서 선택한 번호의 설문을 보여준다.
  3) 막대그래프 출력해서 보여주기
