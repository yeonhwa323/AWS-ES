KICHEN
[1]  주방, 생활 메인 페이지 
<상위>
1. 주방, 생활 카테고리는 ALL/NEW/청소,수납/쿡웨어/테이블웨어/욕실/반려동물 로 구성한다.
2. 메인페이지는 브랜드 별로 이벤트 세일중인 테마를 중심으로 구성하도록 하여 소비자들에게 이벤트에 대해 홍보한다.
4. 브랜드 테마 클릭시
   상위엔 세일중인 테마 이름, 테마 설명, 브랜드 세일기간, 브랜드 대표사진,
   브랜드 이름, 브랜드 설명, 브랜드 홈 바로가기 버튼으로 구성하도록 한다.
   하단엔 브랜드의 베스트순 / 전체 / 주방,생활 / 가구,인테리어 카테고리별 버튼이 있다.
   카테고리를 선택하면 여러 상품들을 보여주도록 한다. 
 5. 상품들은 
   상품사진 / 브랜드이름 / 상품이름 / 할인율 / 가격 / (쿠폰있을경우)쿠폰 / 좋아요 / 리뷰갯수를
   한 묶음으로 상품 가로 4개씩  세로 쭉 나열되어 한눈에 상품에 대한 정보를 알아보도록 화면을 구성한다.
   좋아요는 로그인한 상태인 경우에 누를수 있도록 설정한다.

[2] 주방, 생활 카테고리 선택 후 페이지
<좌측화면>
1. 카테고리들을 한눈에 볼 수 있도록 나열한다.
2. 상품 검색 필터 - 상품정보 / 가격대 / 색상 / 브랜드별로 선택하여 원하는 상품을 검색가능하도록 한다.

<우측화면 >
1. 카테고리 마다의 전체 / 세부 카테고리 나열한다.
ex ) 테이블 웨어 -  전체 / 머그컵 / 식기 세트 / 유리컵 ....등
       욕실          -  전체 / 페이스타월 / 발매트 ......등
2. 상품은 한줄에 3개씩 보기 편한 크기로 나열한다.
- 상품 사진 / 브랜드이름 / 상품이름 / 원래가격 할인율 할인된가격 / 좋아요 리뷰 개수 로 구성한다.

[3] 상품 선택시 세부 페이지화면
1. 상단에 카테고리이름 / 세부카테고리 / 상품분류명 이 있고 다른 카테고리로 변경가능하도록 구성한다.
ex ) 주방,생활 > 테이블웨어 > 유리컵
2. 상세 화면은 윗부분에 브랜드사진 / 브랜드명 / 브랜드 홈버튼을 만들어 원할 시 브랜드 상품을 한번에 볼수있는 홈으로 이동할 수 있도록 구성한다.
3. 그 아래로 상품에 대한 정보로  좌측에 상품사진이 있고
 우측엔 
 - 상품명 / 좋아요
 - ***** $개 리뷰보기
 - 원래가격 / 할인율 / 할인가격   /  쿠폰받기버튼
 - 적립률
 - 나의 구매 가능 가격	할인율 / 할인가격 / 옆 화살표버튼
 ( 옆 화살표 버튼 클릭시)
 상품 쿠폰 할인/장바구니 쿠폰 할인/결제수단 할인/ 보유 마일리지
 사항을 확인하고 본인에게 해당하는 사항을 선택하여 할인 받을 수 있다.
 - 무이자 할부 /  카드사별 할부 혜택 안내
 - 배송정보 / 일반 출고 등 출고일정 안내
 - 배송비  / 배송비 안내 (~원이상 무료배송 , 추가배송비 )
 - 상품 선택사항 ( 사이즈, 컬러 등)
 - 장바구니 담기버튼 / 바로 구매 버튼 으로 구성한다.
4. 상품정보 다음부분엔 브랜드 알림사항, 상품세부설명, 상품정보보기, 판매자정보를 게시한다.
5. 그 밑으로 관련 상품 5개씩 4페이지로 구성되어 있다.
6. 리뷰는 먼저 포토리뷰 사진들을 나열하고
- 별점 / 닉네임 dus****         /      날짜
- 옵션 : ~~~                      /      리뷰 작은사진
- 리뷰글 의 형식으로 구성한다.
7. 상품 Q&A
- ALL / 상품문의 / 재입고문의 / 사이즈문의 / 배송문의 / 기타 / Q&A 쓰기 로 카테고리를 만들어 원하는 카테고리의 상품 Q&A를 볼수 있도록 한다. 상품문의 형식은 [상품문의] 비밀글입니다.  / 닉네임 wis*** / 날짜 / yes or no (문의완료여부) 와 같이 구성한다.
8. 배송정보, 교환, 환불, A/S 안내가 게시되어 있다.
9. BEST 상품은 5개씩 10페이지 - 50위까지 순위를 매긴다.  실시간 / 일간 / 주간 / 월간 으로 나눈다.
10. 다른 고객이 함께 구매한 상품 ( 5개씩 4페이지)
11. 관련 이벤트
이렇게 한 화면으로 상세페이지가 구성되어져있다.


결제 / 환불
[1] 결제
상품 화면에서 상품을 골라 결제를 진행하면
1. 좌측화면엔 배송 정보, 쿠폰사용 및 상품정보, 마일리지, 결제방법, 혜택사항이 뜨게 된다.
1) 배송 정보     * 표시는 필수입력 항목
기존 배송지 / 신규입력
- 배송지명
- 수령인 *
- 배송지 *
- 연락처1 *
- 연락처2
- 기본배송지로 등록 체크박스
- 배송시 요청사항 선택사항 으로 구성되어있다.
2) 쿠폰 사용 및 상품 정보 / 총 수량표시
- 상품 정보 / 상품 쿠폰 & 장바구니 쿠폰 존재 여부 로 구성되어있다.
3) 마일리지
- 사용금액 입력 / 사용가능마일리지, 보유마일리지 로 구성되어있다.
4) 결제방법
- 신용&체크카드 / 토스페이 / 네이버페이 / 카카오페이 / 삼성페이 / 페이코 / 무통장입금 등 선택
- 카드 선택시 카드사 선택 항목 으로 구성되어있고 결제 창으로 넘어가게된다. 
   무통장입금은 출금계좌정보를 입력하게 되어있다.
5) 혜택사항
ex) [카카오페이] 일광전구 최대 5% 즉시할인 ... 등 으로 구성되어있다.

2. 우측화면엔 결제금액, 주문 관련 동의사항, CHECK OUT 버튼으로 구성되어있다.
1) 결제금액
-총 상품 금액  ~원
-쿠폰할인 금액         ~원
  ㄴ 상품 쿠폰         ~원
  ㄴ 장바구니 쿠폰   ~원
-마일리지 사용         ~포인트
-배송비                 +3000원
-총 결제 금액          ~원 으로 구성되어있다.
2) 주문 사항 관련 동의
- 개인정보 수집/ 이용 동의..등 체크박스로 구성되어있다.
3) CHECK OUT 버튼 클릭시 결제창으로 넘어가게 된다.

3. 결제창은 카드 또는 기타 페이 선택할 경우 뜨게 된다.
1) 결제창은 이용약관 동의 / 약관보기 ,  카드 / PAYCO 선택 ,
   상품명 / 상품가격 / 결제 금액 으로 구성되어 있고 결제를 진행하면 된다.
2) 결제 완료시 
주문이 완료 창은
- 주문번호
- 주문상품정보
    ( 상품 / 상품정보 / 수량 / 진행상태 : 결제완료 ) 로 구성
- 결제정보
   ( 결제방법 / 주문상태 / 주문접수일시 / 결제완료일시 / 배송비 / 결제금액 / 영수증 출력버튼 으로 구성)
- 배송정보
   ( 받으시는 분 / 휴대폰번호 / 주소 / 배송요청사항 으로 구성 ) 
- 계속 쇼핑하기 / 주문&배송조회 버튼 으로 구성되어 있다.

[2] 취소, 환불
[마이페이지에서 나의 쇼핑정보 주문배송조회에서 상품 선택하여 취소/환불을 진행할 수 있다.
총 3단계로 구성한다.

- STEP 1. 취소상품 선택

상품정보 / 진행상태 / 수량 선택하고 선택완료버튼을 눌러 다음 단계로 넘어간다.

- STEP 2. 취소사유 작성
취소 상품 정보를 확인하고 취소 사유 작성한 뒤 다음 버튼을 눌러 다음단계로 넘어간다.
- STEP 3. 취소 정보 확인
취소 상품 정보를 확인하고 취소 / 환불 정보를 다시 확인한다.
결제금액 (상품금액 / 배송비/ 쿠폰할인) 환불예정금액(마일리지 환불 / 신용체크카드환불금액 ) 로 구성
취소 환불 정책 설명으로 구성되어 취소/환불이 마무리된다.

[3] 결제 / 취소,환불 조회
결제는 마이페이지의 나의쇼핑정보>[주문배송조회]에서 확인할 수 있도록 한다.
취소,환불은 마이페이지의 나의쇼핑정보>[취소/교환/반품 내역]에서 취소 환불 사항 확인할 수 있도록 한다.
취소, 환불 카테고리는 ALL / 주문취소 / 상품교환 / 반품접수 로 구성한다.

[취소/교환/반품 내역]은 CS처리구분 / 주문번호 / 접수제목 / 접수일자 / 진행상태 / 완료일자 로 구성한다.
